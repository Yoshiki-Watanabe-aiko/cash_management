## 1. システム概要
個人用および個人事業主用の複数口座・クレジットカード・証券口座を統合管理するローカルWebアプリケーション。完全ローカル環境で動作し、情報漏洩リスクを排除しつつ、高度なデータ分析と自動化を実現する。

## 2. 開発環境・技術スタック
- **OS**: Windows
- **IDE**: Visual Studio Code
- **AI Agent**: Claude Code
- **バージョン管理**: Git（プロジェクト固有の運用は[`rules/git.md`](rules/git.md)、グローバル規約は`~/.claude/GIT_CONVENTIONS.md`）
- **インフラ/実行環境**:
  - Windowsタスクスケジューラ（日次バッチ実行用、深夜2時想定）
  - ホストバインド (`0.0.0.0`) による同一ローカルネットワーク内からのアクセス許可（認証は設けない。利便性優先）
- **データベース**: PostgreSQL（Windowsローカルに直接インストール済みの環境を利用）。マイグレーション管理はAlembic。命名規則・DDL詳細は[`詳細設計書/03_データベース設計.md`](詳細設計書/03_データベース設計.md)を参照
- **バックエンド**: Python 3.14.4 / FastAPI（依存関係管理: uv、ロギング: 標準`logging`、データ処理: pandas、祝日計算: jpholiday）
- **フロントエンド**: React 19 + Vite + TypeScript + TailwindCSS v4 + shadcn/ui（ルーティング: react-router-dom、データフェッチ: TanStack Query、テーブル: TanStack Table、チャート: Recharts）
- **通知連携**: Discord（Webhook）。バッチ実行の都度、機関ごとの成功/失敗サマリーを送信

## 4. データ管理方針
- マスタテーブルは接頭辞`m_`、トランザクション（実績・履歴）テーブルは接頭辞`t_`を付与する（グローバル規約`~/.claude/CLAUDE.md`と同一方針）。
- 全テーブル・カラムに日本語コメント（`COMMENT ON TABLE`/`COMMENT ON COLUMN`）を付与する。
- ドメイン用語の定義は[`../CONTEXT.md`](../CONTEXT.md)を参照。

## 5. 非機能要件
- **環境変数**: `.env`ファイルでPostgreSQL認証情報・Discord Webhook URL・Gmail用アプリパスワード（個人用・事業用）・MFログイン情報・口座名義文字列・長期休暇期間を管理（Git管理外）。命名規則はグローバル規約`~/.claude/ENV_CONVENTIONS.md`、実際の項目一覧は[`../.env.example`](../.env.example)を参照。
- **CORS設定**: FastAPI側でフロントエンドからのアクセスを許可するCORSミドルウェアを設定する。
- **アクセス制限**: 簡易的なアクセス制限は不要とし、利便性を優先する（ローカルネットワーク内限定）。
- **バックアップ**: `pg_dump`による日次自動バックアップ（直近30日保持＋各月末分は1年保持）。詳細は[01_データ収集パイプライン設計.md](詳細設計書/01_データ収集パイプライン設計.md)を参照。
- **テスト方針**: [`rules/testing.md`](rules/testing.md)を参照。
